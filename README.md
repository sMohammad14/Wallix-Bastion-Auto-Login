## WALLIX Bastion Auto Login Extension

As the name suggests, this extension intelligently logs you in whenever login is required, using the login URL, username, and password you provide.

## Why Intelligent?

Because when you log out, it won't try to log in again until you open a new page or refresh the same page. Also, when the session expires and re-authentication is needed, it will attempt to log in again.
